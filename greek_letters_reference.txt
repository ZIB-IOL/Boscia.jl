Greek Letters Quick Reference
============================

Common Greek letters used in this codebase:

Lowercase:
α (alpha) - regularization parameters
β (beta) - regression coefficients  
θ (theta) - angles and parameters
λ (lambda) - Lagrange multipliers
μ (mu) - strong convexity parameters
ϵ (epsilon) - small constants

Uppercase:
Ω (Omega) - matrices/constants

Windows Alt Codes (hold Alt + number):
α: Alt + 224
β: Alt + 225
θ: Alt + 233
λ: Alt + 235
μ: Alt + 230
ϵ: Alt + 238
Ω: Alt + 234

Copy-paste these letters directly into your code:
α β θ λ μ ϵ Ω

Usage examples from the codebase:
- α = 1.3  # regularization parameter
- μ = 1.0  # strong convexity
- θ = 1/2  # sharpness exponent
- ϵ = 1e-3 # small constant
- Ω = rand(rng, Float64) # matrix constant
